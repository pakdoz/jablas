Situs Cepat dengan Redis Cache & Global CDN by Stackpath

Kombinasi dua fitur ini akan membantu situs Wordpress anda loading cepat dan ringan!

Global CDN

Dengan CDN, situs Wordpress anda akan cepat diakses dari manapun, baik dari dalam maupun luar negeri; dan meringankan kinerja server. CDN server tersebar di 165 datacenter di seluruh dunia, dan unlimited bandwidth untuk semua klien! Powered by Stackpath.

Redis Cache

Redis Cache akan melakukan cache database, sehingga jika pengunjung mengunjungi situs anda, tidak harus melakukan query ke database lagi, karena akan dilayani oleh Redis melalui cache yang sudah dibuat. Sehingga loading Wordpress akan lebih cepat!
